<!DOCTYPE html>
<html>
  <head>
		<title>My tips page</title>
		<meta charset = "utf-8">
		<link rel="stylesheet" href="css/reset.css">
		<link rel="stylesheet" href="css/style.css">
	</head>
  <body>
  </body>
</html>

This is a basic HTML template I use. All the work with all the tasks starts from this small template.

Doctype is just html. It means that we use the latest version of the HTML that is v.5 now.

Inside links we have stylesheet with href="css/reset.css". In this css we abandone some styles of the browser. Be aware that we abandone just the most common and weird styles that I don't use or resetup inside my own css files. It can be a reason of unexpected behaviour of some tags if you didn't use this file before.
